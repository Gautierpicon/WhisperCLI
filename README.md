# WhisperCLI

Discuss with ChatGPT through your terminal


## Features

- Prompt chatgpt directly in a terminal
## Run Locally

Clone the project

```bash
git clone https://github.com/Gautierpicon/WhisperCLI
```

Go to the project directory

```bash
cd WhisperCLI
```

Create a virtual environment:
```bash
python -m venv venv
```

Activate the virtual environment:
```bash
source venv/bin/activate  # On Linux/Mac
# or
venv\Scripts\activate     # On Windows
```

Install requirements

```bash
  pip install -r requirements.txt
```

Run WhisperCLI

```bash
  python main.py
```
## Environment Variables

To run this project, you will need to add the following openrouter environment variable to a .env file:

```bash
  API_KEY=your_api_key_here
```
